JSX Basics & Rules

Syntax Extension for JavaScript - Looks like HTML, powers React.

src/components/Todo.jsx
01

Single Root Element

Return a single parent. Use Fragments <>...</> to group multiple elements without adding extra nodes to the DOM.

02

Close All Tags

Every tag must be closed. Self-closing tags like <img /> and <input /> are mandatory in JSX.

03

camelCase Attributes

Use JS property names. class becomes className, onclick becomes onClick.

JSX expressions can be embedded using curly braces {}.

export default function TodoApp() {
  const todos = ['Learn JSX', 'Build Component', 'Ship App'];

  return (
    // Rule 1: Use Fragment for single root
    <>
      <div className="container mx-auto"> {/* Rule 3: className */}
        
        <h1 style={{ color: '#00bf53' }}>My Task List</h1>
        
        <ul className="list-disc">
          {todos.map((item, index) => (
            <li key={index}>
              Check item: {item}
            </li>
          ))}
        </ul>

        // Rule 2: Self-closing tag
        <input type="text" placeholder="Add new..." />
      
      </div>
    </>
  );
}
