Do's & Don'ts

Prefer Functional Components

Use modern functions with Hooks over class-based components for cleaner, reusable code.

Keep State Minimal

Avoid redundant state. Lift state up when needed, but prevent prop drilling via Context.

List Rendering & Keys

Always use unique, stable key props for lists to ensure efficient DOM updates.

Type Safety & Quality

Use TypeScript or PropTypes. Enforce quality with ESLint and React Testing Library.

Performance

Implement code splitting with React.lazy and styling with Tailwind or CSS Modules.
