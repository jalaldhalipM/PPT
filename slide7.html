Hooks: useState & useEffect

Managing state and side effects in functional components.

React 16.8+

The Two Rules

Top Level Only: Don't call Hooks inside loops, conditions, or nested functions.

React Functions Only: Call Hooks from React function components or custom Hooks.

useState

Adds local state to function components.

const [state, setState] = useState(initial);

useEffect

Performs side effects (data fetching, DOM updates).

useEffect(() => { ... }, [dependencies]);

import { useState, useEffect } from 'react';

export default function Counter() {
  // 1. Initialize state
  const [count, setCount] = useState(0);

  // 2. Side effect: update document title
  useEffect(() => {
    document.title = `Count: ${count}`;
  }, [count]); // Dependency array

  return (
    <div className="p-4">
      <p>You clicked {count} times</p>
      
      <button onClick={() => setCount(count + 1)}>
        Increment
      </button>
    </div>
  );
}
