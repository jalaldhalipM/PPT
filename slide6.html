State: Local, Dynamic Data

Internal component memory that can change over time and trigger re-renders.

What is State?

Private & Local

State is fully encapsulated within the component. Other components cannot access it unless passed as props.

Triggers Render

Unlike local variables, updating state notifies React to re-render the component and update the UI.

Counter Example 


import { useState } from 'react';

function Counter() {
  // Declare state: [currentValue, updateFunction]
  const [count, setCount] = useState(0);

  return (
    <button onClick={() => setCount(count + 1)}>
      Clicked {count} times
    </button>
  );
}
Pattern: Lift state up to share between components
Anti-Pattern: Storing computed data (derive it instead)
