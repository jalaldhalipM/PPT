Props: Passing Data Down

The mechanism for passing data from parent to child components, making them dynamic and reusable.

Key Characteristics

Unidirectional

Data only flows one way: down from Parent to Child. Children cannot modify props.

Read-Only

Props are immutable. To change UI, parents pass new props, triggering a re-render.

Configurable

Makes components reusable by allowing different data to be passed to the same component structure.

Functions Too

You can pass functions (callbacks) as props to allow children to communicate back to parents.

Usage Example App.js

function App() {
return (
<Button
label="Save"
onClick={handleSave}
/>
);
}

function Button({ label, onClick }) {
return <button onClick={onClick}>{label}</button>;
}
